# 📊 AI Resume Analyser

An AI-powered resume analysis tool built with Python, Streamlit & Google Gemini API.

## 🚀 Features
- 📄 PDF & Image resume upload
- 🤖 AI-powered analysis using Gemini API
- 📊 ATS Score & detailed scoring breakdown
- ✅ Strengths & improvement suggestions
- 🏢 Recruiter mode — screen multiple candidates
- 🌍 Remote job search links
- 📥 PDF report download
- 👤 Experience level based analysis (Fresher to Senior)

## 🛠️ Tech Stack
- Python
- Streamlit
- Google Gemini API
- PyPDF2
- Pytesseract (OCR)
- ReportLab (PDF generation)

## ⚙️ How to Run
1. Clone the repo
2. Install dependencies:
   pip install streamlit PyPDF2 requests reportlab pillow pytesseract
3. Run the app:
   streamlit run App.py
4. Enter your Gemini API key in the sidebar
